# Omakotitalon sähkönkulutuksen infografiikka

Web-pohjainen työnäyte, joka visualisoi 10 päivän omakotitalon sähkönkulutusta.
Tarkoitus: data → selkeä tarina → kevyt animaatio.

## Suunniteltu datalähde
- Tapo-älypistorasiat
- Home Assistantin energiamittaus (CSV/JSON)

## Julkaisu
Tämä repo on tarkoitus julkaista GitHub Pagesin kautta.
